# Задание 1

### Текст задачи

Сформировать список из чисел, противоположных вводимым значениям.

### Алгоритм решения

1. Запрашиваем у пользователя ввод строки с числами, разделенными пробелами
2. Разбиваем строку на отдельные элементы
3. Преобразуем каждый элемент в целое число с проверкой корректности ввода
4. Создаем новый список, применяя к каждому элементу функцию изменения знака (x -> -x)
5. Выводим исходный и результирующий списки на экран

# Задание 3

### Текст задачи

Создайте собственные функции для выполнения основных операций над списками (добавление/
удаление/поиск элемента, сцепка двух списков, получение элемента по номеру).

### Алгоритм решения

1. Добавление элементов

addToBeginning - добавление элемента в начало списка.

addToEnd - добавление элемента в конец списка с использованием конкатенации.

2. Удаление элементов

removeFirst - удаление первого вхождения элемента (рекурсивный обход списка)

removeAll - удаление всех вхождений элемента (с использованием фильтрации)

removeAtIndex - удаление элемента по индексу (разделение списка на две части)

3. Поиск элементов

findIndex - поиск индекса первого вхождения элемента (рекурсивный обход)

contains - проверка наличия элемента в списке

4. Сцепка списков

concatenate - объединение двух списков (использует оператор @)

5. Получение элемента по индексу

getAtIndex - получение элемента по индексу с обработкой ошибок

tryGetAtIndex - безопасная версия, возвращающая Option
